# Anmol Srivastava - Personal Portfolio

Welcome to the repository for my personal portfolio website! This site serves as a digital showcase of my journey as a full-stack developer, highlighting my technical skills, featured projects, and background. 

## ✨ Features

* **Interactive UI/UX:** Smooth animations and a responsive theme design for an engaging user experience.
* **Integrated Resume Viewer:** Built-in functionality to directly view my latest resume without leaving the site.
* **Background Audio:** Seamless background music integration to complement the browsing experience.
* **Project Showcase:** Detailed breakdowns of my development work and API integrations.

## 🛠️ Tech Stack & Skills

While this portfolio highlights my frontend design capabilities, my broader technical stack spans full-stack development and API engineering:

* **Languages:** JavaScript, Python, SQL
* **Environments & Frameworks:** Node.js, Flask, Deno
* **Backend & Deployment:** Supabase, Render
* **AI & API Integration:** Claude API, Google Gemini, Shivaay

## 🚀 Featured Projects Highlighted on the Site

* **MUJ Navigator:** An interactive Progressive Web App (PWA) built specifically for campus navigation at Manipal University Jaipur.
* **Veritas:** An AI-powered pre-consultation clinical history collection tool that utilizes the Claude API to gather patient symptoms in a structured format.
* **Analyze-Waste:** A practical full-stack application utilizing Python and JavaScript ecosystems.

## 💻 Local Setup

To run and modify this portfolio locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/anmolsrivastava073/portfolio.git
   ```
2. Navigate into the project directory:
   ```bash
   cd portfolio
   ```
3. Open the project in your preferred code editor. You can use an extension like Live Server to view it, or spin up a local development server depending on your build tool.
